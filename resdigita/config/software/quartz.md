Quartz.resdigita.com est notre site internet de documentation, principalement technique sur l'ensemble des dispositifs en lien avec le numérique des Grands Voisins.

Ce site utilise le logiciel libre de génération de site statique [Quartz]() dans un entrepôt géré sur [github:chris2fr/quartz](https://github.com/chris2fr/quartz).  Le contenu est dans des fichiers de format Markdown (une forme de format texte avec des conventions pour un traitement de mise en forme). Pour la gestion du contenu, j'utilise Obsidian dans le dossier "content" de l'entrepôt. 

La génération du site est faite par moments.
